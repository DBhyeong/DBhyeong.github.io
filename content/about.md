> **AI 시대에 '도메인'을 가진 데이터 분석가** — 게임·이커머스·회계 도메인을 직접 코드로 분석하고, 코딩 에이전트·LLM 워크플로로 분석·검증·정리까지 자동화하는 12년차.

## Hyeong

안녕하세요, 데이터 분석가이자 디지털 마케터 **Hyeong**입니다. 광고홍보학을 전공한 비전공 출신으로 데이터에 입문해 12년째, "문제가 벌어지는 현장"을 이해하고 그것을 **직접 코드로 분석·자동화**하는 일을 해왔습니다.

SEO에서 출발해 이커머스 가격비교, 게임 3사 라이브 데이터 분석, 퍼포먼스 마케팅을 거쳐, 지금은 데이터 파이프라인·업무 자동화·콘텐츠/SEO를 맡고 있습니다. 한 분야에 머무르기보다 **도메인을 옮겨 다니며 "그 바닥의 핵심 지표가 무엇인지"를 정의하고 코드로 푸는 것**을 즐깁니다.

제 무기는 두 가지입니다 — **도메인 지식 + 자동화 실행력**. 그리고 2026년부터는 여기에 **AI를 도구로 써서 결과를 내는 실행력**을 더하고 있습니다. "AI를 만드는 사람"이 아니라, "AI를 업무에 붙여 더 빨리·정확히 결과를 내는 사람" 쪽입니다.

## 요즘 하는 일 — AI 워크플로 자동화 (2026~)

- **다중 에이전트 오케스트레이션** — 리서치·리뷰·정리를 여러 에이전트로 병렬화하고, 수집=결정론적 코드 / 판단·검증=LLM으로 분리(maker≠checker)하며, 1차 출처로 적대적 팩트체크합니다.
- **ai-news-digest** — 의존성 0 Node로 RSS/Discourse를 모아 큐레이션·팩트체크하는 개인 AI 뉴스 파이프라인.
- **평문 Markdown LLM 지식볼트** — 벡터DB 없이 수만 개의 파일을 LLM이 읽는 위키로 정리(자격증명·개인정보 자동 레닥션 포함).
